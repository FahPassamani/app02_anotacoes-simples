Voce foi contratado para criar um app de anotacoes simples, contendo 2 abas:
- A primeira com o campo de anotacoes.
- A segunda com a visualizacao das anotacoes salvas.

Dados:
- Devem haver 2 abas, cada qual com um Fragment.
- A primeira aba permitira criar e salvar as anotacoes.
- A segunda aba permitira visualizar as anotacoes salvas.

Requisitos / Regras do negocio:
- Fornecer Feedback as acoes do usuario.
- Realizar salvamento via SharedPreferences.
